┌──────────────────────────────────────────────────────────────────────────┐
│                    ACTIVATION DES LOGS (debugging)                       │
└──────────────────────────────────────────────────────────────────────────┘

☐ 1. Éditer le fichier wp-config.php (via FTP ou cPanel)

☐ 2. Chercher : define('WP_DEBUG', false);

☐ 3. Remplacer par :
      define('WP_DEBUG', true);
      define('WP_DEBUG_LOG', true);
      define('WP_DEBUG_DISPLAY', false);

☐ 4. Sauvegarder le fichier

☐ 5. Soumettre un formulaire de test

☐ 6. Consulter /wp-content/debug.log

☐ 7. Chercher les lignes commençant par [FSS]

☐ 8. Une fois le problème résolu : remettre WP_DEBUG à false

AUCUNE LIGNE [FSS] = Le plugin ne s'exécute pas (vérifier activation)

┌──────────────────────────────────────────────────────────────────────────┐
│                    DÉPANNAGE EXPRESS                                     │
└──────────────────────────────────────────────────────────────────────────┘

PROBLÈME : Aucun email n'est envoyé
☐ Vérifier qu'au moins 1 email dans "Email Addresses"
☐ Tester l'envoi WordPress (plugin Check Email)
☐ Consulter les logs [FSS]

PROBLÈME : Tous les emails vont au même destinataire
☐ Vérifier qu'il y a PLUSIEURS adresses dans la liste
☐ Supprimer et recréer la liste (réinitialisation)
☐ Consulter les logs

PROBLÈME : Le routage thématique ne marche pas
☐ Vérifier que "Thematic Filter Mode" = Enabled
☐ Vérifier que le bon champ est sélectionné
☐ Soumettre 1 formulaire avec chaque valeur AVANT de configurer
☐ Consulter les logs pour voir la valeur détectée

PROBLÈME : Les emails arrivent en spam
☐ Installer WP Mail SMTP
☐ Configurer un service SMTP professionnel (SendGrid, Mailgun, etc.)
☐ Vérifier les DNS (SPF, DKIM)

│                    BONNES PRATIQUES                                      │
└──────────────────────────────────────────────────────────────────────────┘

☑ Toujours configurer une liste de fallback (même avec routage thématique)

☑ Tester AVANT production (avec vraies adresses)

☑ Documenter la configuration (capture d'écran + document écrit)

☑ Utiliser des emails génériques (equipe@) plutôt que personnels (marie@)

☑ Surveiller les logs les 2 premières semaines

☑ Prévoir la gestion des absences (procédure écrite)

☑ Vérifier régulièrement que tout fonctionne (1×/mois)

☑ Garder ce document à portée de main

│                    GLOSSAIRE ÉCLAIR                                      │
└──────────────────────────────────────────────────────────────────────────┘

ROTATION        Distribution tour à tour (A → B → C → A...)

THÉMATIQUE      Catégorie basée sur un champ (Santé, Retraite...)

FALLBACK        Liste de secours si routage échoue

CC              Copie envoyée à TOUS les formulaires

FIELD ID        Numéro du champ Formidable (ex: 8)

INDEX           Position actuelle dans la rotation (0, 1, 2...)

WP_DEBUG        Mode debug WordPress (logs dans debug.log)

[FSS]           Préfixe des logs du plugin

SMTP            Protocole d'envoi d'emails (SendGrid, Mailgun...)
